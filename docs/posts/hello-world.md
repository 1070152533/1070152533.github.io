这是第一篇博客文章，记录使用 **VitePress + GitHub Pages** 搭建本博客的过程。

## 技术栈

- **VitePress**：基于 Vite 和 Vue 3 的静态站点生成器
- **GitHub Pages**：免费托管静态网站
- **GitHub Actions**：自动构建与部署

## 为什么选择 VitePress

1. 构建速度极快，基于 Vite
2. 支持在 Markdown 中使用 Vue 组件
3. Markdown 文件即路由，简单直观
4. 内置搜索、暗色模式等功能
